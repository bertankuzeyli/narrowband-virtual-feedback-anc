# Experimental Narrowband ANC: Virtual Feedback on STM32H7

This repository contains an experimental implementation of a Single-Channel Narrowband Active Noise Cancellation (ANC) system built on the STM32H7 architecture. Designed to explore the physical and algorithmic challenges of real-time acoustic signal processing, the project utilizes a Feedback Filtered-x LMS (FxLMS) adaptive algorithm. By leveraging CMSIS-DSP alongside hardware-level features like DMA-driven I2S double-buffering and the hardware FPU, the system aims to achieve the microsecond-level latency required for active noise suppression. However, translating textbook DSP theory into a functional physical environment presents ongoing challenges, making this repository a continuous work-in-progress focused on stability and acoustic phase alignment.

The defining architectural focus of this project is the implementation of an Internal Model Control (IMC) topology, which synthesizes a 'Virtual Reference' signal to eliminate the need for a physical reference microphone. By mathematically subtracting the estimated secondary path effects from the total measured acoustic error, the algorithm attempts to isolate the true physical noise. This approach is designed to overcome the critical phase-synchronization (phase-lock) failures commonly experienced with synthetic feedforward designs. While the underlying mathematical model is sound, fine-tuning the physical acoustic setup to prevent feedback loops (howling) and ensure algorithmic convergence remains the primary focus of current development.
